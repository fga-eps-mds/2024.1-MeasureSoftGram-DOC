# Agile Earned Value Management (AgileEVM)

<div align="justify">&emsp;&emsp;
A técnica de Earned Value Management (EVM) é uma forma de acompanhar um projeto integrando três aspectos fundamentais: escopo, custo e tempo. Ela parte de um escopo bem definido e, com base nisso, permite medir quanto do trabalho planejado já foi realmente entregue, quanto foi gasto e se o projeto está dentro do prazo.
</div>

<div align="justify">&emsp;&emsp;
Sendo assim, surge o Agile Earned Value Management (AgileEVM) — uma adaptação do EVM para metodologias ágeis, como o Scrum. Nessa versão, em vez de depender de um planejamento rígido, são utilizadas métricas típicas do ágil, como story points e velocity, tornando o acompanhamento mais compatível com a natureza iterativa do projeto.
</div>

> Explicação do AgileEVM foi adaptada da documentação do [MeasureSoftGram 24.1](https://fga-eps-mds.github.io/2024.1-MeasureSoftGram-DOC/planejamento/custos/).
